代码块功能验收样例

Created2025-11-16|Updated2025-11-20|技术
|Word Count:657|Reading Time:2mins|Post Views:

本文档用于全面检验博客主题的代码块功能，特别是在 PJAX 动态加载后，各项功能是否依然完好。

主要检验点：

  1. 多种语言的 语法高亮 是否正确。
  2. 鼠标悬浮时 “复制”按钮 是否出现。
  3. 点击“复制”按钮后，内容是否能被 完整、正确地 粘贴到记事本中。
  4. 带有 标题 或 行号高亮 的高级功能是否正常。

1. 基础代码块

这是一个没有指定语言的普通代码块，通常会以纯文本样式显示。

2. 带语法高亮的语言

JavaScript

Python

Go

YAML (常用于 Hugo 配置)

Shell / Bash

3. 高级功能测试

带标题的代码块

你的主题可能支持为代码块添加标题或文件名。

行号高亮

检查特定行是否能被高亮显示。

4. 验证步骤

  1. 直接访问：直接在浏览器中打开本页面，检查所有代码块的语法高亮是否正常，复制功能是否可用。
  2. PJAX 切换：
    • 先访问博客的主页或者其他任何文章。
    • 然后，通过点击站内链接导航到本页面。
    • 再次检查所有代码块的高亮和复制功能。如果此时依然正常，说明你的 PJAX 集成非常成功！
